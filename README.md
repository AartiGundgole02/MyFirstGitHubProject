# MyFirstGitHubProject

## Overview
This repository contains automation scripts and test scenarios for web applications using **Playwright**, **TypeScript**, and **API testing**. The project demonstrates end-to-end testing, user registration flows, login verification, and more.

## Tech Stack
- **Frontend Automation:** Playwright, TypeScript
- **Backend/API Testing:** REST APIs
- **Testing Tools:** Node.js, npm
- **CI/CD (Optional):** GitHub Actions

## Project Structure
MyFirstGitHubProject/
│
├─ e2e/ # End-to-end test scripts
├─ pages/ # Page object models (if any)
├─ tests/ # Test scenarios
├─ package.json # Node project dependencies
├─ playwright.config.ts # Playwright configuration
└─ README.md



## Features
- User Registration & Login automation  
- CRUD operations test scenarios  
- API endpoint verification  
- Configurable Playwright test setup for multiple browsers

```bash
git clone https://github.com/AartiGundgole02/MyFirstGitHubProject.git

2. Install dependencies:
cd MyFirstGitHubProject
npm install

3. Run tests
npx playwright test

4.View test report (if configured):
npx playwright show-
